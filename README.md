# Automatic-Water-Dispenser Prototype
This is a motor driven system that allows user to access water in a tank by touching a sensor once.

**Materials**

*Arduino Uno*

*Touch pad sensor*

*Wires*

*LED Light*

*PC with the IDE*

*5v motor*

*Water Container*

*Bread board*
